# React_Mooc

A Massively open online course by the University of Helsinki 
the courses teaches programming in Full stack development using primarily
Node.js and React. 

Topics covered 
- React
- Express.js backend
- Testing frontend and backend
- Redux
- React Router
- custom React hooks
- GraphQl
- Tyescript
- React native
- CI/CD
- Containers
- RDBMS
